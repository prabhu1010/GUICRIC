# GUICRIC

A GUI version of cricket simulator

SetIndexes 					: constants for indexes
InitBatterData				: takes in a batsman and clears his stats
SendOpeners 				: calls InitBatterData for incoming batsmen
SetFooter					: footer data for that innings
InitializeCenterBoard 		: For the active innings

Initialize					: variables used in scoring an innings

BatterScores				: When run is scored
UpdateFaced					: after each legal ball
HandleExtras				: for wide or legbye
WicketFallen				: for incoming guy
